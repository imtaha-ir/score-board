# Score Board

Score Board is a beginner-friendly frontend learning project built with React, TypeScript, and Vite. It is designed as a practical example of building a real-world web application from scratch, covering the full journey from initial setup to deployment preparation.

## Course Context

This project is part of a frontend video course that teaches modern web development fundamentals in a hands-on way. The course covers the essential steps for creating a complete frontend application, including:

- project setup with Vite
- building reusable UI components with React
- managing state and user interactions
- adding styling and improving user experience
- preparing the app for production build and deployment

## Project Overview

Score Board is a simple scoreboard application for tracking points in games, classrooms, competitions, or live events. It demonstrates how a small app can grow from a basic idea into a polished interactive interface.

## Features

- Add players with custom names
- Increase or decrease scores with one-click controls
- Prevent duplicate player names
- Reset scores or remove all players
- Use color-coded score cards for clear visual tracking
- Enable fullscreen mode for presentations and live displays

## Technical Stack

- React
- TypeScript
- Vite
- ESLint

## What Students Learn

By completing this course and project, students will gain practical experience with:

- creating a React application from scratch
- using TypeScript for safer and more maintainable code
- managing component state and events
- building simple but interactive UI flows
- preparing a project for production using build tools
- understanding the basics of frontend development workflow

## Learning Outcome

After finishing this course, students should be able to move from beginner-level knowledge to a solid introductory level in frontend development and confidently build small real-world applications.

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
