# Score Board

A sleek and simple scoreboard app built with React, TypeScript, and Vite. It is designed for quick score tracking in games, classrooms, competitions, or live events.

## Features

- Add players with custom names
- Increase or decrease scores with one-click controls
- Reset scores or remove all players
- Color-coded score cards for easy visual tracking
- Fullscreen mode for presentations and live displays

## Tech Stack

- React
- TypeScript
- Vite

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
